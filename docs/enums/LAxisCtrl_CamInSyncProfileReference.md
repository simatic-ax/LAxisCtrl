# CamInSyncProfileReference

| Symbol | Value | Description |
| ------ | ----- | ----------- |
| IN_ADV_DYN_PARAMS | 0 | Synchronization in advance using dynamic parameters |
| IN_ADV_LEAD_VAL_DIST | 1 | Synchronization in advance using leading value distance |
| DIRECT_IMMEDIATELY | 2 | Direct synchronous setting |
| SUBSEQ_LEAD_VAL_DIST | 3 | Subsequent synchronization using leading value distance |
| SUBSEQ_LEAD_VAL_DIST_AS_OF_CURR_POS | 4 | Subsequent synchronization using leading value distance starting from current leading value position |
| DIRECT_AT_END_OF_CAM | 5 | Direct synchronous setting at the end of the cam |
| IN_ADV_LEAD_VAL_DIST_AS_OF_CURR_POS | 6 | Synchronization in advance using leading value distance starting from current leading value position |
| DIRECT_AT_LEAD_VAL_POS | 7 | Direct synchronous setting at specific leading value position |
