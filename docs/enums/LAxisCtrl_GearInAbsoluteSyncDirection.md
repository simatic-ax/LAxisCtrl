# GearInAbsoluteSyncDirection

| Symbol | Value | Description |
| ------ | ----- | ----------- |
| POSITIVE | 1 | Positive direction The following axis may only travel in positive direction during synchronization. |
| NEGATIVE | 2 | Negative direction The following axis may only travel in negative direction during synchronization. |
| SHORTEST_DIST | 3 | Shortest distance Changes in direction are permitted for the following axis during synchronization. |
