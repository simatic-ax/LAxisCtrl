# JogMode

| Symbol | Value | Description |
| ------ | ----- | ----------- |
| CONTINUOUS_JOGGING | 0 | Continuous jogging |
| INCREMENTAL_JOGGING | 1 | Incremental jogging new increment (starts with new increment) |
| INCREMENTAL_JOGGING_CONTINUOUS | 2 | Incremental jogging continue increment (continue the last increment if not completed) |
