# OffsetDirection

| Symbol | Value | Description |
| ------ | ----- | ----------- |
| POSITIVE | 1 | The leading value distance is in the positive direction of motion of the leading value. |
| NEGATIVE | 2 | The leading value distance is in the negative direction of motion of the leading value. |
| CURRENT | 3 | The leading value distance is in the current direction of motion of the leading value. |
